Moved Files:
------------
Android Gradle projects use a different directory structure than ADT
Eclipse projects. Here's how the projects were restructured:

In Android SDK:
* AndroidManifest.xml => androidSDK/src/main/AndroidManifest.xml
* res/ => androidSDK/src/main/res/
* src/ => androidSDK/src/main/java/
In ThroneExample:
* AndroidManifest.xml => throneExample/src/main/AndroidManifest.xml
* assets/ => throneExample/src/main/assets/
